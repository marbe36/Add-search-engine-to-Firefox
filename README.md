# Add search engine to Firefox
Add manualy additional search engines to Firefox on desktop and mobile. 

Supported Search Engines: 

1. Startpage - Startpage (Android), Startpage (Desktop)
2. These methods can be used for any other Search Engine by replacing link to Search Engine: Kagi, Ecosia, DuckDuckGo, Brave, Baidu, Yandex and others.

# Add Startpage to Firefox on Android 
Make Startpage your default search engine on Firefox mobile (Android)

Copied from: https://support.startpage.com/hc/en-us/articles/4520868432788-Make-Startpage-your-default-search-engine-on-Firefox-mobile-Android go to this website for more actual version


1. Tap the three vertical dots next to the URL bar.

2.  Tap Settings.
<img width="480" height="2340" alt="image" src="https://github.com/user-attachments/assets/08c3d385-a396-4867-8790-4898433b9f7b" />

3.  Tap Search.
<img width="480" height="2340" alt="image" src="https://github.com/user-attachments/assets/2ce450ef-c7a2-4474-99be-08aee8f09d3e" />

4.  Tap Add search engine.
<img width="480" height="2340" alt="image" src="https://github.com/user-attachments/assets/d577d0be-e342-4ac7-b2b7-1eccd1fc81ec" />

5.  Tap Other and fill in the two fields and tap the check mark:

Data to fill:

Search string to use: https://www.startpage.com/sp/search?q=%s

Search suggestion URL (optional): https://www.startpage.com/osuggestions?q=%s

# Add Startpage to Firefox on Desktop
## Firefox extension method
[Guide on Startpage site](https://support.startpage.com/hc/en-us/articles/4480404660628-Make-Startpage-your-default-search-engine-in-Firefox)

## Fast method
1. Go to https://www.startpage.com.
2. Right-click on Startpage link, and press on Add "Startpage Search".
<img width="503" height="349" alt="image" src="https://github.com/user-attachments/assets/1cc21c28-94c3-4f0a-83f3-e0beedc56bb6" />

3. Press the three vertical bars next to the URL bar.
<img width="1378" height="1129" alt="image" src="https://github.com/user-attachments/assets/6b6505a7-6829-44a0-9fbc-50cd58f93789" />

4. Press Settings.
<img width="1283" height="1045" alt="image" src="https://github.com/user-attachments/assets/62a645c2-ef0f-4a61-9561-1fe68cda0daf" />

5. Press Search.
<img width="1284" height="1047" alt="image" src="https://github.com/user-attachments/assets/72c7569c-da6c-419c-8d88-d247309d5ce5" />

6. Press on default search engine and choose Startpage as default search engine.

## Manual method (for some search engines)
1. Press the three vertical bars next to the URL bar.
<img width="1378" height="1129" alt="image" src="https://github.com/user-attachments/assets/6b6505a7-6829-44a0-9fbc-50cd58f93789" />

2. Press Settings.
<img width="1283" height="1045" alt="image" src="https://github.com/user-attachments/assets/62a645c2-ef0f-4a61-9561-1fe68cda0daf" />
